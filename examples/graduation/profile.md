# Example Profile: Dissertation Workflow

- Project label: Dissertation completion
- Timezone: America/New_York
- Morning planning time: 08:30
- Evening check-in time: 23:00
- Weekly review time: Sunday 20:00
- Reminder style: direct
- Default focused-time target: 3h baseline + 2h stretch
- Minimum acceptable focused time: 2h on recovery days
- Output preferences:
  - Interactive HTML workbench: yes
  - Desktop wallpaper PNG: yes
  - Markdown report: yes

## Reporting Preferences

- Focused work includes writing, analysis, experiment design, reviewing outputs, and deciding next actions.
- Agent-delegated work includes long-running coding, refactoring, testing, artifact generation, and summarization.
- Agent work must be reviewed or converted into a concrete artifact before it counts as progress.
- The system should directly call out secondary projects that crowd out dissertation-critical work.
