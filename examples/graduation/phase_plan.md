# Example Phase Plan: Dissertation Workflow

## North Star

Complete a dissertation first draft before the spring graduation deadline.

## Phase 1: Re-baseline, close old manuscript, compress core analysis

- Dates: Month 1
- Primary outcome: remove the nearly finished manuscript from the main workload and make the core analysis chapter draftable.
- Required outcomes:
  - Manuscript A is submission-ready, submitted, coauthor-ready, or blocked by a named external dependency.
  - Core analysis code reaches a reviewable state with one validated path.
  - Main analysis chapter has a written skeleton and figure slots.
  - Dataset workflow status is documented enough to support the next month.
  - Secondary framework projects stay capped.
- Gate:
  - End of Month 1: readable chapter skeleton, main result path, figure set v0, and remaining-revision checklist.

## Phase 2: Core evidence and committee package

- Dates: Months 2-3
- Primary outcome: make the core dissertation chapters committee-ready.
- Required outcomes:
  - Manuscript A is no longer a daily burden.
  - Core analysis has figures, robustness checks, interpretation, and draft v1.
  - Dataset workflow is frozen enough to support chapter statistics.
  - Committee package shows completed prior chapter, dataset chapter progress, and core analysis results.
- Gate:
  - End of Month 3: committee-ready package for the dissertation core.

## Phase 3: Optional final technical chapter go/no-go

- Dates: Months 4-5
- Primary outcome: decide whether an optional modeling chapter is full chapter, short chapter, appendix, or future work.
- Required outcomes:
  - Task definition and baseline are completed before advanced model work expands.
  - Optional chapter cannot delay core chapter revisions.
- Gate:
  - End of Month 5: final go/no-go decision.

## Phase 4: Full draft assembly

- Dates: Months 6-7
- Primary outcome: integrate all chapters into a complete first draft.
- Required outcomes:
  - Full chapter drafts exist.
  - Code and data inventory exists.
  - Committee feedback path is clear.
